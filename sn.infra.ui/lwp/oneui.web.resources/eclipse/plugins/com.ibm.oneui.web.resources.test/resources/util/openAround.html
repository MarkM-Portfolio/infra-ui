Menu utility

Opens a dijit pointed by menuId (typically a dijit.Menu) in a popup placed around the node in event.target.

Typically called on a onclick event in the page, es: <a onclick="com.ibm.oneui.util.openAround('id of your menu', this, null, event)">Click me</a>

Use dojo.connect to bind it programmatically to a DOM node.

Clicking here will pop up a menu with the chosen orientation.
