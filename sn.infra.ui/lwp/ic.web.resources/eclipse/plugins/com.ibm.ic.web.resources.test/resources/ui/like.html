Like widget

Module ID: ic-ui/Like
JSDoc

The signature Like widget used throughout Connections. It is backed by a dojo.data API compliant data store. It has three operating modes, anonymous, editable, and read only, all of them with or without a popup. The popup shows a list of people who liked the item. Callers can also choose to “unbox” it and show it without the gray background.
