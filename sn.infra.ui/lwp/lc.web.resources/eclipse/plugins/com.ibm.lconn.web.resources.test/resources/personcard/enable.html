The person card and sametime proxy awareness are controlled by the following configuration options:

Legacy person card (3.0) is enabled when:

  • The 'profiles' service is enabled in LotusConnections-config.xml
  • The generic property 'com.ibm.lconn.personcard.enable' in LotusConnections-config.xml is set to 'legacy' or is not specified.
Note: to complete disable the person card, set the generic property 'com.ibm.lconn.personcard.enable' to 'off' in LotusConnections-config.xml

Standard person card (4.0) is enabled when:

  • The 'profiles' service is enabled in LotusConnections-config.xml
  • The generic property 'com.ibm.lconn.personcard.enable' is set to 'standard'

Sametime Awareness via the client is enabled when:

  • The <sametimeAwareness> tag in profiles-config.xml is set to enabled

Sametime Proxy Awareness is enabled when:

  • The <sametimeAwareness> tag in profiles-config.xml is disabled
  • The 'sametimeProxy' service in LotusConnections-config.xml is enabled
