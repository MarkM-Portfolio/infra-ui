Language Selector

Description Example Code
1. Using lconn.core.header.enableLanguageSelector(). You should see a link that pops up a menu with three language options, plus Browser Setting. 
new lconn.core.LanguageSelector(
    dojo.byId('languageSelector2'),
    {"en":"English","it":"Italiano","iw":"\u05e2\u05d1\u05e8\u05d9\u05ea"}
    "lcLang",
    {path: '/', expires: -1, domain: 'localhost'}
);
2. Using constructor. You should see a link that pops up a menu with three language options, plus Browser Setting. 
new lconn.core.LanguageSelector(
    dojo.byId("languageSelector2"),
    {"en":"English","it":"Italiano","iw":"\u05e2\u05d1\u05e8\u05d9\u05ea"}
    "lcLang",,
    {path: '/', expires: -1, domain: 'localhost'}
);
3. Single language. You should see no link, because it makes no sense to let users choose when there's only one choice available. 
new lconn.core.LanguageSelector(
    dojo.byId("languageSelector3"),
    {"en":"English"},
    "lcLang",,
    {path: '/', expires: -1, domain: 'localhost'}
);
4. Force language via LotusConnections-config.xml. You should see no link, although there are two options: a language set by an admin, and BD (browser-determined). 
lconn.core.header.enableLanguageSelector(
    "languageSelector4",
    {"en":"en","BD":"BD"},
    "lcLang",
    "",
    -1
);
