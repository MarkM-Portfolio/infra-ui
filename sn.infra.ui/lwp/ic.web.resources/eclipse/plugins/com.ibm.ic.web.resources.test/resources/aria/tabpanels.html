Tab Panel

Module ID: ic-ui/aria/TabPanel
JSDoc

This helper enables a tab panel for keyboard navigation. It acts as a single tab stop when focusing in e.g. from this link. Tabs can be focused using the arrow keys. The caller owns displaying the associated panel on click.
